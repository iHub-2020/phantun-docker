Critical Safety Information
  • Phantun creates TUN interface and uses FakeTCP to disguise UDP traffic.
  • Client Mode requires MASQUERADE iptables rules (auto-enabled).
  • Server Mode requires DNAT iptables rules (auto-enabled).
  • Without iptables - Phantun relies on kernel's native processing and is less stable.
  • MTU must be reduced by 12 bytes (TCP header - UDP header).
